goproxy.parspooyesh.com

A global proxy for go modules.

First, you will need to enable the Go Modules feature and configure Go to use the proxy.

Bash:

export GOPROXY=https://goproxy.parspooyesh.com

Now, when you build and run your applications, go will fetch dependencies via goproxy.parspooyesh.com.

Github

Note: this proxy can't fetch your private repos of course.
